목적

이 실습은 임의의 개수로 존재하는 동일한 타입의 원소들을 연속된 공간에 저장하기 위해 가변 배열을 사용하는 연습을 합니다. 이때, 자유영역(heap 영역)에 저장 공간을 할당하는 new 연산자와 배열의 사용이 종료되면 자유영역(heap 영역)의 저장 공간을 해제하는 delete 연산자의 사용을 연습합니다.



설명

프로그램이 실행 중에 배열의 크기를 알 수 있다면 동적으로 배열을 생성해야 합니다. 이때, 배열의 저장 공간을 컴퓨터 메모리의 스택 영역이나 힙 영역에 생성할 수 있습니다. 예를 들어, 힙 영역에 가변 배열의 저장 공간을 생성하기 위한 방법은 다음과 같습니다.



int SIZE = 0;

cin >> SIZE;

int* arr = new int[SIZE];



문제

N의 길이를 갖는 정수형 배열이 주어진다. 다음으로 해당 배열의 i 번째 원소의 값을 인덱스로 사용하여 해당 인덱스에 해당하는 배열의 값을 출력하는 프로그램을 작성하시오.

단, new 연산자를 사용하여 배열의 저장공간을 할당 받고, 프로그램 종료 전에 delete 연산자로 할당 받은 저장공간을 해제하시오.



제출파일

main.cpp

15.csv





입력

첫째줄에 정수의 개수 N이 주어지며, 두 번째 줄에 N개의 정수가 주어진다.

세 번째 줄에 i가 주어진다.

1 <= N <= 100 



출력

arr[ arr[i] ] 값을 출력한다.



입출력 예제

입력	출력
5
1 4 2 3 1 
1	1
3
2 1 1
2             	1               
6
5 4 2 5 4 0
5	5
